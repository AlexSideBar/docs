Get up and running with Alex Sidebar in just a few minutes. Follow these simple steps to enhance your Xcode development experience.

## Installation

<Steps>
  <Step title="Download">
    Download the latest DMG from our [releases page](https://github.com/DanielEdrisian/AlexSideBar-Public/releases).
  </Step>
  
  <Step title="Install">
    Open the downloaded DMG file and drag Alex Sidebar to your Applications folder.
  </Step>
  
  <Step title="Launch">
    Launch Alex Sidebar from your Applications folder. It will automatically integrate with Xcode.
  </Step>
</Steps>

## Optional Configuration

<AccordionGroup>
  <Accordion title="API Key Configuration">
    While Alex Sidebar works out of the box with a free tier, you can configure your own API keys for supported Large Language Models (LLMs) to unlock extended usage or specific models.
    
    Detailed instructions for API key setup can be found in the [AI Integration](/ai-integration) section.
  </Accordion>

  <Accordion title="Model Configuration">
    Alex Sidebar supports various LLM options to suit your needs:
    
    - Choose from different cloud-based models based on performance and cost
    - Set up local models using Ollama for offline usage
    - Configure model parameters to optimize for your use case
    
    Visit the [Model Configuration](/model-configuration) section to learn more.
  </Accordion>
</AccordionGroup>
