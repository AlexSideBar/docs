## Pin Chat Input to Bottom

Keep the chat input field fixed at the bottom of the window for easy access and improved usability.

<CardGroup cols={2}>
  <Card title="Always-visible Input Area">
    The chat input field remains visible at all times, ensuring immediate access to the chat functionality.
  </Card>
  <Card title="Quick Access">
    Direct access to chat functionality without scrolling or searching for the input area.
  </Card>
  <Card title="Improved Conversation Flow">
    Seamless interaction with the chat interface, maintaining natural conversation rhythm.
  </Card>
  <Card title="Better Space Utilization">
    Optimizes the display of long chat histories while keeping the input accessible.
  </Card>
</CardGroup>

## Show Hints

Displays helpful tooltips and suggestions throughout the interface to help you:

- Discover keyboard shortcuts
- Access contextual help

<Note>
  You can always toggle hints off once you are familiar with Alex Sidebar's features.
</Note>
