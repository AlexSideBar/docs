## Using Commands

<Steps>
  <Step title="Access Commands">
    Type `@` or click the + button in the chat interface to see available commands.
  </Step>

  <Step title="Search">
    Use the search bar at the top to filter available commands and find what you need quickly.
  </Step>
</Steps>

<Note>
  Commands help you efficiently access resources and context without leaving your chat workflow.
</Note>
