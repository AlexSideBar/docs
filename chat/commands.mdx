## Using Commands

<Steps>
  <Step title="Access Commands">
    Type `@` or click the + button in the chat interface to see available commands.
  </Step>

  <Step title="Search">
    Use the search bar at the top to filter available commands and find what you
    need quickly.
  </Step>
</Steps>

<Note>
  Commands help you efficiently access resources and context without leaving
  your chat workflow.
</Note>

## Command Examples

<Steps>
  <Step title="File Reference">
    ```
    @file MyViewController.swift
    ```
    References a specific file in your codebase
  </Step>

{" "}

<Step title="Documentation Search">
  ``` @docs UITableView ``` Searches Apple documentation for UITableView
</Step>

  <Step title="Model Switch">
    ```
    @model claude-3
    ```
    Switches to Claude 3 model
  </Step>
</Steps>
