## Essential Keyboard Shortcuts
| Shortcut | Action | Description |
|----------|--------|-------------|
| ⌘ + L | New Chat with Code | Start a new AI chat with your selected code |
| ⌘ + K | Inline Completions | Get quick inline code suggestions |
| ⌘ + / | Switch AI Models | Toggle between different AI models |
| ⌘ + N | New Empty Chat | Create a fresh chat without code context |
| ⌘ + ⇧ + L | Add to Current Chat | Append selected code to current chat |
| ⌘ + ⌫ | Stop Generation | Immediately stop the current AI response |
| ⌘ + P | Advanced Search | Open the advanced search panel |
| ESC | Close Panel | Universal close button for any open panel |

## Search Features
- **Advanced Search (⌘ + P)**
  - Search through codebase
  - Filter by file type
  - Search chat history
  - Jump to specific files

## Best Practices
1. **Code Context**
   - Select relevant code before starting chat
   - Include imports and related types
   - Provide clear, specific questions

2. **Model Selection**
   - Use Claude for explanations
   - Use GPT-4 for complex refactoring
   - Try Gemini/Grok for SwiftUI
   - Switch models if not getting desired results

3. **Chat Organization**
   - Pin important chats
   - Use clear titles
   - Group related questions
   - Save useful prompts

## Common Use Cases & Workflows

### Code Review
1. Select code → ⌘ + L
2. Ask for review
3. Add context with ⌘ + ⇧ + L if needed
4. Use /copy to save suggestions

### Quick Fixes
1. Place cursor at error
2. ⌘ + K for suggestions
3. Use ESC to dismiss if needed
4. /retry if suggestion isn't perfect

### Documentation
1. Select function/class
2. ⌘ + L
3. Ask for documentation
4. Specify format (DocC, inline, etc.)

## Configuration Tips
- Customize shortcuts in Settings
- Set default AI model
- Configure auto-completion behavior
- Set up custom prompts for repeated tasks

## Common Pitfalls to Avoid
- Don't provide too little context
- Don't ignore error messages
- Don't forget to include necessary imports
- Don't stick with one model if struggling

## Window Management

- Pin window position in settings

## Pro Tips

- Combine shortcuts for efficiency
- Save frequently used prompts
- Keep chat context focused
- Use image-to-code for UI replication
