## Core Features

<CardGroup cols={2}>
  <Card 
    title="Command+L to Chat" 
    icon="message-bot"
    href="/chat/interface#quick-actions"
  >
    Select any text in Xcode or supported browsers (Chrome, Arc) and instantly start a chat about it. Get immediate AI assistance for code understanding, web content analysis, and solutions.
  </Card>

<Card
  title="Image to Code (Drag & Drop)"
  icon="image"
  href="/chat/image-to-code#features"
>
  Transform designs into code. Simply drag any image into the sidebar and let
  Alex generate the corresponding code for you.
</Card>

<Card
  title="Semantic Codebase Search"
  icon="magnifying-glass"
  href="/search/advanced-search"
>
  Search your entire codebase semantically. Find exactly what you are looking
  for by describing it in natural language and hitting tab.
</Card>

  <Card 
    title="Command+K for Quick Actions" 
    icon="bolt"
    href="/completions/inline-suggestions#using-inline-completions"
  >
    Generate in-file suggestions instantly. Get smart code completions and improvements without breaking your flow.
  </Card>
</CardGroup>
