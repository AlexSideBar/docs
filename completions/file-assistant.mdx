## File Assistant ⚡

File Assistant is a powerful feature that lets you quickly modify files using natural language commands. Just press **Command + Shift + M**, describe what you want to do, and File Assistant will suggest changes instantly.

## Using File Assistant

<Steps>
  <Step title="Open File Assistant">
    1. Open the file you want to modify
    2. Press **Command + Shift + M**
    3. The File Assistant panel appears
  </Step>

  <Step title="Describe Changes">
    Type your request in natural language:
    ```
    "Add a UUID property and make it Codable"
    "Create a function to handle user authentication"
    "Add documentation comments to this class"
    ```
  </Step>

  <Step title="Review & Apply">
    - Review the suggested changes
    - Use Quick Apply (⏩) for instant application
    - Or click Apply to see the diff view first
  </Step>
</Steps>

  <Accordion title="Documentation">
