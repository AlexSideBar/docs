## Quick Code Generation

Alex Sidebar provides powerful inline code completion capabilities to help you write code faster and more efficiently.

  <Card title="Command + K" icon="bolt">
    Trigger in-file suggestions instantly. Get AI-powered completions based on

  <Card title="Tab Completion" icon="arrow-right">

    suggestion.

## Using Inline Completions

<Steps>
  <Step title="Trigger Suggestions">
    Position your cursor where you want to generate code and press **Command +
    K**
    <Note>
      The AI analyzes your current file context to provide relevant suggestions
    </Note>
  </Step>

  <Step title="Review Options">
    - Use arrow keys to cycle through multiple suggestions - Press Enter to
    accept the current suggestion - Press Esc to dismiss suggestions
    <Frame>
      <img
        src="/images/inline-completion.png"
        alt="Inline code completion in action"
      />
    </Frame>
  </Step>
</Steps>

## Tab Completion

<AccordionGroup>
  <Accordion title="Real-time Suggestions">
    As you type, Alex Sidebar provides intelligent code suggestions:
    - Variable names based on context
    - Method completions
    - Import statements
    - Common coding patterns
  </Accordion>

  <Accordion title="Using Tab Complete">
    1. Start typing code normally
    2. When suggestions appear:
       - Press Tab to accept suggestion
       - Continue typing to filter options
       - Use arrow keys to select alternatives
    3. Press Esc to dismiss suggestions
  </Accordion>
</AccordionGroup>
