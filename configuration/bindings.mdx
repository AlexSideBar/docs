## Core Shortcuts

<CardGroup cols={2}>
  <Card title="Code Context" icon="code">
    - **Command + Shift + L**: Add selected lines to context
    - **Command + L**: Create new chat with selected lines
  </Card>

  <Card title="Quick Actions" icon="bolt">
    - **Command + K**: Inline completion
    - **Command + P**: Advanced Search Panel
  </Card>
</CardGroup>

## Additional Features

<CardGroup cols={2}>
  <Card title="Code Suggestions" icon="wand-magic-sparkles">
    - **Tab**: Complete code suggestions (beta)
    - Intelligent code completions based on context
  </Card>

  <Card title="Issue Resolution" icon="bug">
    - Autofix Issues functionality
    - Automatic error detection and resolution
  </Card>
</CardGroup>

<Note>
  These keyboard shortcuts are designed to enhance your coding workflow. The
  most frequently used commands are: - **Command + L** for starting new chats -
  **Command + Shift + L** for adding context - **Command + K** for inline
  completions
</Note>

<Tip>
  Start with mastering these core shortcuts before exploring additional
  features. They cover the most common use cases in your development workflow.
</Tip>
