## Fill Available Width

With this setting, Alex Sidebar will automatically expand to use the available screen width on the side of Xcode, while keeping everything at a comfortable reading size.

## Match Height with Xcode

This setting matches the sidebar to look like it is _almost_ part of Xcode.
